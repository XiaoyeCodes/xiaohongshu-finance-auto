# 配图 Prompt 示例

## 示例 1：问答式配图（副业主题）

```
An infographic poster for Xiaohongshu titled "下班后做什么能多赚 3000？这 5 个副业靠谱" in large, clean Chinese characters. The style is modern friendly flat vector art on a soft beige background.

顶部区域（核心问题）：一个可爱的卡通人物下班后看着手机思考，头顶有对话框"下班后做什么能多赚 3000？"，旁边有个钱袋图标和问号，背景是办公室和家的切换场景。

中部区域（5 个副业）：五个横向排列的卡片，每个卡片有图标和标题。从左到右：1."知识付费"图标：笔记本电脑 + 课程视频；2."自媒体写作"图标：键盘 + 文章页面；3."技能接单"图标：设计工具 + 钱包；4."电商无货源"图标：购物车 + 包裹；5."线下兼职"图标：日历 + 课时费。每个卡片下方标注收入范围"6000/月""8000/月""5000+""3000-8000""2400/月"。

底部区域（收入增长）：一个向上的折线图展示"2020 年：0"→"2021 年：800/月"→"2022 年：3500/月"→"2023 年：8000/月"→"2024 年：12000/月"，折线末端有个开心的卡通人物举着"副业=多一条路"的牌子。

All text in clean correct Chinese. Premium educational design with soft macaron colors. --ar 3:4
```

## 示例 3：清单式配图（财务自由主题）

```
An infographic poster for Xiaohongshu titled "财务自由需要多少钱？我算了笔账" in large, clean Chinese characters. The style is modern friendly flat vector art on a soft beige background.

顶部区域（核心问题）：一个可爱的卡通人物坐在书桌前，拿着计算器和笔记本，思考气泡里显示两个数字"1000 万❌"被划掉和"300 万✅"打勾，头顶有个灯泡表示顿悟。对话框写着"被动收入 > 日常支出"。

中部区域（三个阶段）：三个垂直递进的方框，用向上箭头路径连接。左侧（绿色主题）：1."安全感"图标：盾牌上写着"6 个月生活费 3-5 万"。中间（蓝色主题）：2."选择权（进行中）"图标：打开的门，写着"被动收入 5000/月"。右侧（金色主题）：3."真正自由"图标：沙滩椅 + 太阳，写着"被动收入 2 万/月"。

底部区域（真实数据）：一个干净的进度条展示旅程："25 岁：0"，"26 岁：200/月"，"27 岁：1500/月"，"28 岁：3800/月"，虚线箭头指向"明年目标：5000/月"。结尾处有个开心的卡通人物举着小旗子。小图标展示收入来源：基金分红、课程销售、利息。

All text in clean correct Chinese. Premium educational design with soft macaron colors. --ar 3:4
```

## Prompt 编写要点

### 1. 开头 - 整体说明
```
An infographic poster for Xiaohongshu titled "[标题]" in large, clean Chinese characters.
The style is modern friendly flat vector art on a soft beige background.
```

### 2. 分段描述
- **顶部区域**：核心问题/吸引点
- **中部区域**：核心内容/步骤/对比
- **底部区域**：数据/结果/总结

### 3. 具体图标描述
- ❌ 不要：只说"钱"
- ✅ 要：说"手把硬币放入保险箱，标签 3-6 个月闲钱"

### 4. 文字内容
- 所有文字用中文
- 包含具体数据（金额、百分比、时间）
- 包含金句/总结语

### 5. 结尾规范
```
All text in clean correct Chinese. Premium educational design with soft macaron colors. --ar 3:4
```

### 6. 视觉规范关键词（必须包含）
- 扁平化插画
- 矢量艺术
- 2D 极简风
- 结构化布局
- 马卡龙色系
- 柔和色调
- 干净色块
- 无渐变
- 治愈系财经风
